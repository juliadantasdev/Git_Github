Git-Github

Primeiramente nós baixamos o Git, depois configuramos ele colocando o comando git config --global user.name "juliadantasdev", nesse comando você coloca seu nome de usuario do github, e o outro é git config --global user.email "julia.dantas62@aluno.ifce.edu.br", onde coloca o email que está logado no seu github. Criamos um repositório no github, copiamos o link quando se cria. Voltamos para o Git, depois de configurado, procuramos a pasta que queremos colocar no repositório, clicamos com o botão direito e em seguida em git bash, no comando colocamos git init, para ver o status da pasta, podemos colocar git status e aparecerá de vermelho as "pastinhas" que irão subir, depois nós damos um git add ., e ao olharmos o status com o git status, as "pastinhas" estarão verdes. Em seguida colocamos o comando git remote add origin https://github.com/juliadantasdev/Git_Github.git.
